# Badminton App

A simple and efficient badminton tournament management application.

## Features

- **Tournament Management** - Create and manage badminton tournaments
- **Player Tracking** - Register and track player statistics
- **Match Scheduling** - Automatic match scheduling and bracket generation
- **Score Recording** - Log match results in real-time
- **Leaderboards** - View player rankings and tournament standings

## Installation

```bash
git clone <repository-url>
cd badminton-app
npm install
```

## Usage

```bash
npm start
```

## Requirements

- Node.js 14+
- npm or yarn
